Editor's Note

Vercel automated 58% of PRs in their Next.js monorepo. That's 400+ weekly PRs with agents reviewing and merging without humans in the loop.

This isn't a demo. This is production infrastructure at scale, and it's working. What gets me isn't the percentage — it's that they trust agents enough to merge code into one of the most-used JavaScript frameworks on earth.

The tooling layer is exploding. Smarter model routers (Windsurf), faster inference (Cursor), multi-model validation (GitHub). But the Replit piece hit different: PMs using agents to keep prototypes as source of truth. I'd never thought about using AI to make throwaway code less throwaway.

The timeline estimates are getting nuts. 30% full AI R&D automation by 2028? Six months ago that number was 15%. Either this researcher is drunk on hype, or we're accelerating way faster than anyone expected.
