Editor's Note

Two patterns jumped out today. First, the infrastructure is getting real — OpenAI's security playbook isn't aspirational anymore, it's "here's how you actually deploy Codex agents in production." Vercel's conversation history SDK solves the unglamorous problem of state persistence across platforms. These aren't demos, they're the boring reliability work that lets you ship.

Second, everything is breaking at once. Linux vulnerability disclosure timelines. Age verification pushing down to the OS level. Traditional economic metrics losing predictive power to GitHub commits. The infrastructure is maturing while the rules of engagement are dissolving.

ProgramBench caught my attention because all frontier models fail spectacularly at recreating programs from specs. Either we've hit a fundamental wall in reasoning capability, or the benchmark is testing something orthogonal to how engineers actually work. Most coding isn't clean room implementation, it's adapting existing patterns.
